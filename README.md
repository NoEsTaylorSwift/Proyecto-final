# Agente de Reconocimiento Facial Web - Mi Rostro

Esta versión permite registrar tu rostro desde la cámara y luego buscarlo entre varias personas.

## Funcionamiento

1. Iniciar cámara.
2. Registrar mi rostro.
3. Buscarme.
4. Si el sistema reconoce tu rostro, ejecuta una acción:
   - Mostrar mensaje.
   - Abrir sitio web.
   - Reproducir canción.

## Importante

- El registro del rostro se guarda en el navegador usando localStorage.
- No se suben fotos a ningún servidor.
- Si abres el proyecto en otro teléfono o navegador, debes registrar tu rostro nuevamente.
- Para usar música, agrega un archivo llamado `cancion.mp3`.

## Recomendaciones para que reconozca mejor

- Registra tu rostro con buena luz.
- Toma las muestras de frente.
- Para buscar en multitud, usa la cámara trasera.
- No te alejes demasiado.
- Evita sombras fuertes o demasiadas personas muy lejos.

## GitHub Pages

Para usar la cámara en el celular, sube los archivos a GitHub Pages y abre el enlace HTTPS.
