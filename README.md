# Agente Facial Web con Menú

Proyecto web con dos modos:

## 1. Encontrarme a mí

Sirve para que cualquier persona use la app.

Flujo:

1. Escribir nombre.
2. Iniciar cámara.
3. Registrar rostro.
4. Buscar persona.

## 2. Encontrar a Mariela

Modo limpio para presentación.

Flujo:

1. Iniciar cámara.
2. Presionar "Buscar a Mariela".
3. La app usa el perfil fijo guardado en `perfiles.js`.

## Importante

Para que el modo "Encontrar a Mariela" funcione sin registrar cada vez, el archivo `perfiles.js` debe tener pegado el perfil exportado de Mariela.

## Archivos

- `index.html`
- `style.css`
- `app.js`
- `perfiles.js`
- `README.md`

## Música

Si quieres reproducir canción, agrega un archivo llamado:

`cancion.mp3`

## Recomendaciones

- Usa buena iluminación.
- Para buscar entre varias personas, usa cámara trasera.
- No estés demasiado lejos.
- Prueba antes de exponer.
