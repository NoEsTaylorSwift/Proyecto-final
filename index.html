Proyecto web con cámara

Agente de Reconocimiento Facial

Esta versión registra tu rostro y luego intenta encontrarte entre varias personas. Los demás rostros se muestran como desconocidos.

Primero inicia la cámara, luego registra tu rostro con buena luz. Para buscarte entre la multitud, usa la cámara trasera del teléfono.
Recomendado: 0.50 a 0.55. Menor = más estricto. Mayor = más flexible.

Presiona “Iniciar cámara” para comenzar.

¡Te encontré! Rostro reconocido correctamente.
